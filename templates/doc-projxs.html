ProjXs Documentation

Your comprehensive guide to getting started with ProjXs.

Overview

ProjXs is a versatile app for note-taking and project organization, designed to boost your productivity. With ProjXs, users can:

  • Sign up and securely log in to a personalized workspace.
  • Create and organize projects effectively.
  • Take notes enriched with an AI-powered real-time calculator.
  • Access saved projects and notes conveniently from the sidebar.

Features

1. User Signup and Login

Create an account with an email and password to securely access your workspace.

2. Create New Projects

Start new projects with a title and optional descriptions or notes. Organize your thoughts and tasks seamlessly.

3. Sidebar Navigation

Easily navigate through all saved projects and notes. The intuitive sidebar allows quick access and management.

4. AI-Powered Real-Time Calculator

Perform real-time calculations directly within your notes. Ideal for budgets, planning, or any numerical tasks.

How to Use ProjXs

  1. Sign Up: Visit the signup page and create an account with your email and password.
  2. Log In: Enter your credentials on the login page to access your workspace.
  3. Create a New Project: Click the "New Project" button, add a title, and start writing notes.
  4. Use the Sidebar: View and manage your saved projects and notes from the sidebar.
  5. Real-Time Calculations: Utilize the AI-powered calculator embedded in the notes.

Technical Details

  • Backend: Python with Flask for managing user accounts and data storage.
  • Frontend: HTML, CSS, and JavaScript for a responsive, user-friendly interface.
  • Database: Custom JSON-based database hosted on Google Cloud for efficient data management.
  • AI Feature: Real-time calculation functionality integrated into the note-taking system.
